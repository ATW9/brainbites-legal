LIFE DASHBOARD

Privacy Policy

Last updated: 16 July 2026

Life Dashboard is a private, self-hosted application with a single user — its owner. It is not a commercial product, it has no other users, and it does not offer accounts or sign-ups. This policy explains how it handles data.

1. What it is

Life Dashboard is a personal tracking app (training, body weight, and personal finances) that runs on hardware owned and operated by its owner, in their own home. It is used solely by that individual.

2. WHOOP data

With the owner's explicit authorisation, the app uses the WHOOP API to read the owner's own WHOOP data: recovery score, heart rate variability, resting heart rate, sleep, daily strain, calories burned, workouts, and body measurements (weight).

  • This data is read only for the owner's own account. The app never requests or accesses any other person's data.
  • It is stored in a private database on the owner's own server. It is not uploaded elsewhere.
  • It is used for one purpose: showing the owner their own training and health metrics.

3. Sharing

No data is sold, shared, published, or transmitted to any third party. There is no analytics, advertising, tracking, or profiling of any kind.

4. Storage and retention

Data is held in a local database on the owner's own machine for as long as the owner chooses to keep it. Access tokens are stored privately on that same machine and are used only to refresh the owner's own data.

5. Revoking access

The owner can disconnect the integration at any time from within the app, or revoke access directly in their WHOOP account settings. Doing so stops all further data access immediately.
